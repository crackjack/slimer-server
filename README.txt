Instructions to Install and Run
===================================
-- extract the zip
-- cd into the folder
-- create virutal environment `virtualenv env`
-- activate the virtual environment `source env/bin/activate`
-- install the pip requirements `pip install -r requirements.txt`
-- install the node dependencies through npm `npm install`
-- run the slimer server via node `node slimer_server.js &`
-- run the python script to send request to slimer `python slinode.py`
